## 项目基本说明

### UI说明
	本项目UI使用【hoperun-ui】， 基于element-ui-2.13.1版本进行封装的公司私有UI库，
	具体UI说明文档，请参考element-ui官网 2.13.x的说明
	注意： 使用UI时， 标签el-xxx  均改为 hp-xxx
	
### 项目脚本说明

	npm i: 下载依赖【下载依赖前，请先切换公司私服库地址，然后再切换】
	npm run start: 启动项目
	npm run build: 打包项目
